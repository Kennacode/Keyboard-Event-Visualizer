# ⌨️ Keyboard Event Visualizer

An educational Python application that visualizes keyboard events inside the application window using Tkinter.

## Features

* Real-time key press visualization
* Key classification (Letters, Numbers, Special Keys)
* Timestamp logging
* Total key counter
* Most used key tracking
* Last pressed key display
* Clear history functionality
* Modern dark-themed interface

## Technologies

* Python
* Tkinter
* ttk
* Collections
* Datetime

## How to Run

```bash
python app.py
```
